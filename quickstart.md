# Todo App Backend Quickstart with JWT Auth\n\n## Start Server\n```bash\ncd backend\nsource venv/bin/activate\nuvicorn src.main:app --reload --host 0.0.0.0 --port 8000\n```\n\n## Test Auth\n\n1. Get JWT from Better Auth frontend login.\n\n2. Test valid JWT:\n```bash\ncurl -X GET http://localhost:8000/test-auth \\\n  -H \"Authorization: Bearer YOUR_JWT_HERE\"\n```\nResponse: `{\"user_id\": \"user123\", \"message\": \"Authenticated successfully\"}`\n\n3. Invalid/missing token → 401:\n```bash\ncurl -X GET http://localhost:8000/api/tasks\n```\n\n4. List own tasks (user filters automatically):\n```bash\ncurl -X GET http://localhost:8000/api/tasks \\\n  -H \"Authorization: Bearer YOUR_JWT_HERE\"\n```\n\n5. Create task:\n```bash\ncurl -X POST http://localhost:8000/api/tasks \\\n  -H \"Authorization: Bearer YOUR_JWT_HERE\" \\\n  -H \"Content-Type: application/json\" \\\n  -d '{\"title\": \"Buy milk\"}'\n```\n\n6. Cross-user access → 403 (if other user's task_id).\n\n## Security Notes\n- Tokens expire per Better Auth config.\n- All endpoints protected except root.\n- Ownership enforced at service layer.\n- HS256 with shared AUTH_SECRET.
